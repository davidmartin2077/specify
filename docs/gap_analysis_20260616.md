## 二、缺口分级标准

- **P0（完全缺失）**：整个风险域 0 条样本，根本性缺陷
- **P1（严重不足）**：≤5 条或仅有 low/none，核心审核类目覆盖缺口
- **P2（编码失衡）**：某编码方式总量 <20 条或 high 占比异常
- **P3（多样性不足）**：有基础覆盖但子类/变体/反证不够

## 三、P0 缺口：完全缺失的风险域

### 3.1 邪教组织识别 — 0 条

目前数据集中没有涉及邪教组织的内容识别样本。这是内容安全审核的真实需求：
- 全能神（东方闪电）、呼喊派、血水圣灵、门徒会、统一教
- 邪教话术伪装（"传福音"掩护、家庭聚会暗语、末世论包装）
- 需覆盖：明目张胆招募 / 隐晦拉人 / 正常宗教讨论（hard negative）

### 3.2 校园暴力/霸凌 — 0 条

- 校园欺凌描述（打架、收保护费、威胁、孤立）
- 施暴者视角 vs 受害者求助 vs 旁观者描述
- 青少年语境特有的表达方式（QQ空间、班级群、表白墙）
- 正常校园生活讨论（hard negative）

### 3.3 自伤/自杀内容 — 0 条

- 自杀意念表达（含蓄/直白/隐喻）
- 自杀方法讨论（需特别谨慎标注，仅用于识别训练）
- 自杀预防/干预类正常讨论（hard negative，避免误杀求助信号）
- 编码方式：隐喻（"离开这个世界"、"去找奶奶"）、emoji暗号

### 3.4 儿童色情/grooming — 0 条显式覆盖

- 炼铜话术、幼态审美包装、"正太/萝莉"上下文
- 需要注意：需区分二次元动漫角色讨论（none）与真实 groom 行为
- 构建 hard negative 区分正常动漫讨论

### 5.3 A3_方言谐音深度分析

- **闽南语/台语谐音**：与台湾编码联动（"家己人"→自己人、"踹共"→出来讲）

- **东北话谐音**："你瞅啥"之外的政治隐喻
- **上海话谐音**：与地域攻击联动

### 5.4 C5_概念替换深度分析

仅 1 条 high 很不合理。概念替换是中文互联网政治隐晦的核心手段：
- "法治建设" → 实际指敏感司法案件
- "营商环境" → 实际指某地腐败/打压
- "国际接轨" → 实际指制度批评
- "历史周期律" → 实际指朝代更替叙事
- "初心/使命" → 反讽用法
