# 中国大陆网络用语列表整理
来源：[维基百科：中国大陆网络用语列表](https://zh.wikipedia.org/wiki/中国大陆网络用语列表)
整理日期：2026-06-11

说明：本文件按原页面条目机械整理并压缩释义；部分词语含侮辱、歧视或政治攻击意味，仅作语义辨识参考。

共整理 323 个条目。

## 英文原字或简写
- **Pick**：挑选的意思，指选中最喜欢的人事物。
- **Diss**：Disrespect或者Disparage的简写，在嘻哈文化中指的是一种Rapper用歌曲去发泄不满情绪或攻击对手。

## 汉语拼音缩写
- **NMSL**：“你妈死了”的汉语拼音缩写。
- **YYDS**：“永远的神”的汉语拼音缩写。

## 拟声词
- **滴滴**：拟声词，属于一种较为温和的打招呼方式。

## 缩句词
- **不明觉厉**：虽然不明白是什么，但是感觉好厉害啊；虽然不明白他（你）在说什么，但是感觉很厉害的样子。
- **然并卵**：“然而并没有什么卵用”：常用作某件事物或者某些行为对当前事态的发展不会造成任何的影响或者对某件事物没有任何用处，“卵”指代男性生殖器。
- **画美不看**：“画面太美我不敢看”讽刺所见事物不堪入目。
- **我伙惊/我伙呆**：我和我的小伙伴们都惊呆了。
- **啊痛悟蜡**：啊，多么痛的领悟（点蜡烛）。
- **人艰不拆**：人生已经如此的艰难，有些事情就不要拆穿。
- **累觉不爱**：来自一名豆瓣网友“路西”发言：“马上就要13岁了，单身，身心疲惫（累），感觉不会再爱了”。
- **爷青结**：爷的青春结束了。
- **十动然拒**：（女生）十分感动，然后拒绝了他。
- **百错仍硕**：百余处表述、引用、格式、翻译、分析不规范等错误，却仍被认定合格并授予硕士学位（参见武汉大学图书馆争议事件）。
- **你生我梦**：你的生活我的梦。

## 谐音
- **蓝瘦香菇**：壮语腔汉语「难受想哭」的普通话谐音。

## 政治类 / 中国大陆 / 人群类
- **大白**：对身穿白色防护服的医护人员及防疫相关一线工作人员和志愿者的称呼。
- **50万**：指《公民举报间谍行为线索奖励办法》，其中规定“对防范、制止间谍行为或侦破间谍案件发挥特别重大作用，贡献特别突出的线索，给予10万至50万元奖金。
- **赵家人**：出自鲁迅著名中篇小说《阿Q正传》，一般指中共权贵阶层。
- **精神资本家（精资）**：最早出于鲁迅杂文《“新时代”的避债法》，指对并非资本家、但却乐于维护资产阶级及资本家利益的人或群体的蔑称。

## 政治类 / 中国大陆 / 人群类 / 民间政治群体
- **五毛/网评猿**：贬称，政府官方称之“网络评论员”。
- **美分党/带路党/三毛**：是“五毛”的对立面，指被支持中国大陆政府或中共的人认为受雇在网上发表不利于中国政府、主张所谓“自由民主”和/或有利于台湾、欧美、日韩等政府的言论者。
- **八千党**：指中国大陆内部的反华人士以及大量发布反华反政府言论的人士。
- **自带干粮的五毛（简称“自干五”）**：倾向于中国政府及中国共产党言论、自称立场客观，自发发表言论而非受雇发表的「五毛」。
- **自费美分/自干美**：指亲西方、发布亲西方言论并自称或被认为自发发布言论、没有报酬的人。
- **民国粉/国粉**：指喜爱中华民国、中国国民党或国军的群体，有时为贬义。
- **小粉红**：贬称，原指中国文学网站晋江文学城的女性用户，现指在网络上活跃的朴素麻木“爱国”而忽略或无视各种弊病的年轻群体等。
- **神友/神蛆/神神**：指最早发源于百度贴吧论坛“神奈川冲浪里”吧的部分网民。
- **绿教**：指代伊斯兰教。
- **毛左/左人/网左**：对中国民间互联网中怀念毛泽东时代、不认同中国特色社会主义的左派的称呼。
- **支黑**：是指对中国或中国文化、历史、社会、政治等持强烈负面态度和批评的人群或现象。
- **大陆北方网友/中国北方网友**：指冒充中国大陆网民的境外网络水军。
- **黄俄**：也作“黄鹅”、“俄殖”，对2022年俄罗斯入侵乌克兰中亲俄罗斯网民的贬称，名称来源于黄俄罗斯计划。
- **乌贼**：对2022年俄罗斯入侵乌克兰中亲乌克兰网民的贬称，有时也称为“黄乌”或“乌鸡”（与“黄鹅”对应）。
- **鱿鱼**：对以色列人或支持以色列者的贬称，因“犹”与“鱿”发音相同而得名。
- **小日子过的不错的日本人/小日子**：对日本人、日本亦或其支持者的中性偏贬义称呼。
- **日子人 / 岁静党**：厌恶风险和宏大叙事，只想安于一亩三分地，追求「岁月静好」者。

## 政治类 / 中国大陆 / 人物类
- **习包子**：指现任中共中央总书记习近平，源自习近平吃包子事件。
- **习维尼**：指习近平，因迪士尼卡通角色「小熊维尼」与习近平外表体态相似。
- **常凯申**：指前中华民国总统蒋介石，源于蒋介石姓名的错误翻译，往往带有贬义，随后成为网络上对其的调侃称呼之一，形成历史和军事迷因。
- **腊肉**：对前中国共产党中央委员会主席毛泽东的蔑称。
- **蛋炒饭**：指毛泽东长子毛岸英，源自毛岸英死亡始末的过程争议，不过官方则批评此说法。
- **战略忽悠局局长**：指中国人民解放军国防大学教授张召忠。
- **叼盘侠**：指环球时报前总编辑、网络大V胡锡进，因其胡在许多重大事件的相关评论伴随官方立场摇摆不定、“言论前后矛盾”。
- **夹头**：指知名爱国大V司马南，因其2012年赴美过年时被机场电梯夹住头部，网民借此讽刺其表里不一、“反美是工作、赴美是生活”。
- **钢铁公司**：中性词，指邓小平，因其在1974年首次被毛泽东称为“钢铁公司”，因而部分网友（多左派）称其为“钢铁公司”

## 政治类 / 中国大陆 / 中国共产党/中华人民共和国政府
- **伟光正**：源自习惯性词组“伟大的、光荣的、正确的中国共产党”，亦可以指中共相关人物。
- **改开党/特色党/二共**：民间左翼人士和毛泽东主义者常用词，源于邓小平的“改革开放”以及“中国特色社会主义”。
- **惠台办/跪台办/施舍办**：指国务院台湾事务办公室（国台办），指责其对台工作时只懂得向台湾让利，仅凭一味妥协退让式的绥靖政策就妄图换取尊重，而实际效果甚微。
- **咱妈**：最初指我们的母亲，现大陆网络上常用于指，中国共产党和中华人民共和国政府，尤其从2025年4月贸易战以来，当中华人民共和国的实力获得世界认可，或在外交上采取强硬姿态时，往往会使用这一称呼。

## 政治类 / 中国大陆 / 其他
- **拜登打钱/拜登经费到账/CIA打钱/CIA经费到账**：源于民间在美国总统拜登上台后的一种论调，认为其会加大外宣投入和对中国国内异见人士的扶持力度以制造舆论。
- **警惕拜登打X牌**：最初由中华人民共和国外交部和广泛用于表示警惕美国以某事为借口遏制中国发展，例如陈平在《眉山论剑》中提到要“警惕境外势力打马恩牌”。
- **和谐你全家**：绿坝花季护航的衍生梗。
- **叮咚鸡、大狗叫、袋鼠鸡、见缸马**：源自海南社区工作人员的口音形成的空耳，原句为“听通知”、“戴口罩”、“带手机”、“健康码”，后制作成动画形象，成为疫情时代的象征和后疫情时代的迷因。
- **GFW/墙**：防火长城（又叫中国国家防火墙），平时说的“某网站被GFW了”或“被墙了”是指该被该防火墙所屏蔽。
- **翻墙/科学上网/梯子**：指突破网络审查。
- **茧中互联网**：“简中互联网”的谐音，暗指中国大陆的网民大多如在茧中一般，坐井观天。
- **天朝大局域网/中国特色互联网/Chinternet**：由于中国大陆当今的互联网审查和防火长城的屏蔽，绝大部分大陆网民只能访问中国国内网站，于是诞生此称呼，用来讽刺中国大陆的网络制度。
- **冲塔**：原意是游戏《英雄联盟》中某方冲入对方防御塔的行为，现多用于暗喻主动提及政治敏感话题的行为。
- **大杀器**：大规模杀伤性武器的简称，源于胡戈的网络电影《鸟笼山剿匪记》。
- **连夜绣红旗、火线入党（爱国）等**：形容政治立场的急速转变，对这类极速转变立场者，亦有相对的嘲讽性说法，如1912年入宫、49年入国军等，指其看不清形势。

## 政治类 / 台湾 / 人群类
- **1450**：源自2019年台湾农委会编列的雇用网络小编预算高达1450万新台币，被质疑是雇佣网军。
- **来电了？**：源于台湾在2022年兴达发电厂停机事故中传言的网络流量减少，在中国大陆被用来质疑对热点事件发表评论者是否为“1450”等“对岸网络水军”，或单纯用于辱骂对热点问题持不同意见者。
- **王师**：中华民国国军或美军的称呼，取自陆游诗句“王师北定中原日”（《示儿》），影射中华民国过去光复大陆计划，现也用于影射美国重返亚太对中国施加军事压力事件，带讽刺意味，如“南望王师又一年，王师还剩几个连？
- **旗鱼人/其余人**：因行政院在其官网介介绍台湾人口组成时，称“台湾目前已设户籍人口2.6％为原住民族群，另外来人口占1.2％，其余人口占96.2％”，引发强烈争议。
- **海岛奇兵**：原本是是一款以进攻岛屿、对抗敌对势力为核心玩法的策略游戏，通常表达对蒋介石与国民党政府因在第二次国共内战中战败被迫迁台的讽刺。

## 政治类 / 台湾 / 中国国民党
- **刮民党**：谐音，是亲共产党网民和部分自由派人士对中国国民党的蔑称，该称呼源自国共内战时期中国共产党对中华民国政府和国民党的丑化宣传，意指当时执政的国民党通过恶性通货膨胀等政策搜刮民脂民膏。
- **丐帮党**：蔡英文执政时期对所谓不当党产的处理导致国民党一度财务状况紧张；再加之国民党高层近年来频繁访问大陆、谋取北京方面对台提供经济让利，引发部分对台鹰派网民的不满，与“跪台办”等国台办蔑称相对应。
- **KMT**：国民党的威妥玛拼音首字母缩写和英文简称，进而延伸出“老K”、“烤馒头”等代称。

## 政治类 / 台湾 / 其他
- **湾湾/WW/4V/南波湾/莱克多巴湾/凤梨湾/对岸**：为中国大陆网民对台湾或中华民国的称呼（源自叠字风，中性词）。
- **蛙蛙**：原意青蛙，相比前者“湾湾”更加贬义，常用于指代民主进步党人士，用于形容绿色标识，不用于指代其他党派。
- **来电了？**：源于台湾在2022年兴达发电厂停机事故中传言的网络流量减少，在中国大陆被用来质疑对热点事件发表评论者是否为“1450”等“对岸网络水军”，或单纯用于辱骂对热点问题持不同意见者。
- **没出息**：王世坚质询时任台北市长柯文哲「（台北花了170亿的世大运）本来应该从从容容游刃有余，现在是匆匆忙忙连滚带爬！
- **冻蒜**：源自台湾，台语“当选”的谐音，指希望某人当选某一职位（但在中国大陆多用于调侃），如“蔡英文冻蒜”、“床破冻蒜”。

## 政治类 / 香港 / 人群类
- **黄丝**：指香港民主派、本土派等非建制派阵营，以及支持其立场的人士，因2014年占中期间非建制阵营以黄丝带为象征而得名。
- **忠诚废物**：来源于北京“港澳研究会”理事田飞龙对只讲政治立场却缺乏政策制定与执行能力的香港建制派的评价。
- **蓝丝**：指香港建制派阵营，以及亲政府、亲建制派、撑警者，与“黄丝”相对，因建制派的代表色为蓝色得名，由于「丝」与「尸」在普通话和粤语的读音均类似，因此「蓝尸」被用于蔑称。
- **保皇党**：指香港建制派阵营，往往指责其没有政治立场，只是一味附和北京。
- **左胶**：指香港泛民主派中的左派阵营，包括社民派、托派等，以及支持其立场的人士，因其往往被建制派、中右翼泛民主派、右翼本土派等阵营认为不切实际而得名，亦有用来指责台湾地区的左派。

## 政治类 / 苏联/后苏联国家 / 人物类
- **苏穗宗/玉米晓夫**：指前苏联共产党第一书记赫鲁晓夫。
- **苏勋宗**：指前苏联共产党总书记勃列日涅夫。
- **普大帝/普大弟/晋凉公**：指现任俄罗斯总统普京。

## 政治类 / 美国 / 人物类
- **奥观海**：指前美国总统奥巴马。
- **美伶宗**：指美国前总统里根，因其出身演员故得此名。
- **川建国**：指美国总统川普。
- **美国前总统拜登**：页面列为相关网络用语，未在该行给出更详细释义。
- **美稀宗**：2021年9月21日，美国前总统拜登当天在白宫与英国首相约翰逊会面时，疑似当场腹泻大便失禁，故称。
- **拜振华**：由奥巴马和特朗普的外号延伸而来。
- **哈哈姐**：指2024美国总统候选人贺锦丽。

## 政治类 / 朝鲜半岛
- **想吃广东菜**：这句话是歌词“相思风中开”的空耳，出自歌曲《你若三冬》的戏腔部分，后来被网友做成DJ版在网络流行。
- **此视频拍摄于“夜间”、“被灼烧了/被烫伤了”**：衍生自恩情文学，比喻金正恩所在视频中白天的场景因其自带的“领袖的光芒”“照亮”了周围所形成的“反差感”。
- **XX一定要有XX**：衍生自恩情文学，原为金正恩视察某地说过的一句话。
- **忠！橙！**：韩国军队敬礼时高喊的“忠诚”（충성），是韩国陆军及部分军种最经典、正式的敬礼口号，网络用于常用于对韩国保守派总统的调侃。

## 政治类 / 朝鲜半岛 / 人物类
- **带桶泳**：大韩民国总统（韩语：대한민국 대통령<small>／</small>大韩民国大统领，<small>罗马化：</small>Daehan Minguk Daetongnyeong）的谐音。
- **全小将**：大韩民国总统全斗焕，在电视剧《第五共和国》第一集中，他办公桌上的牌子印的是保安司令部司令官小将全斗焕，但实际上他的军衔是少将。
- **将军/太阳**：指现任朝鲜最高领导人劳动党总书记、国务委员会委员长金正恩。

## 政治类 / 柬埔寨
- **XX疑似有点过于城市化了**：讽刺波尔布特认为城市是资本的象征，故进行了一系列去城市化运动。
- **“老木屋”“坐钢叉”“八足马”“boy特斯拉”“沦落的美”**：柬埔寨歌曲Sat Tee Touy的空耳歌词。

## 政治类 / 柬埔寨 / 人物类
- **波波、波波杀**：波尔布特（或译“波尔波特”）的简称，红色高棉领导人，曾发动红色高棉大屠杀。

## 政治类 / 其他 / 人物类
